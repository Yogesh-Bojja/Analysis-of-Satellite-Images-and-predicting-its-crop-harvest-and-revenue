                        Property Boundaries - DCDB - July 2012 (DNRM) (Deprecated)
                        This layer shows Queensland's Digital Cadastral Database taken as of the 15th July 2012. This layer is now out of date and deprecated by an updated snapshot of the database.

                        Dataset description
                        This polygon layer is a 'lite' version of the Digital Cadastral Database (DCDB) showing minimal attribute data about the property boundaries e.g.: base lot polygons, Lot and Plan attributes and an accuracy statement covering the whole of Queensland. See additional information also.
                        This data is updated nightly on the QSpatial portal, but only periodically on the eAtlas.
